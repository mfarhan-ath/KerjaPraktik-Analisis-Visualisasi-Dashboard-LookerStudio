# KerjaPraktik-Analisis-Visualisasi-Dashboard-LookerStudio
Mengembangkan dan menerapkan visualisasi dashboard dengan menggunakan Looker  Studio untuk memonitor pertumbuhan penduduk di setiap kecamatan. Visualisasi data  interaktif ini memungkinkan pengguna untuk membandingkan jumlah penduduk dari  tahun ke tahun dengan mudah dan memberikan informasi yang mendalam terkait distribusi penduduk.
